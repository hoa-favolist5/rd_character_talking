## ğŸš€ Quick Start

### Prerequisites

- Node.js 20+
- Python 3.11+
- Docker & Docker Compose
- AWS credentials (for Polly, S3)
- Anthropic API key

### Development Setup

1. **Start PostgreSQL:**
```bash
docker-compose up -d postgres
```

2. **Start the API:**
```bash
cd apps/api
python -m venv .venv
source .venv/bin/activate
pip install -e .
uvicorn main:app --reload
```

3. **Start the frontend:**
```bash
cd apps/web
npm install
npm run dev
```
